# Dotfiles for the programs I am using

- This is configured in a way to work with both Windows and Linux

## Linux
- Install `stow`
- Run `cd && git clone https://github.com/marianguceanu/dotfiles && cd dotfiles && stow.` 

## Windows
- No need to install anything here
- Run `cd && git clone https://github.com/marianguceanu/dotfiles && cd dotfiles && ./symlinks_win.ps1`
- The script is meant to provide you only wezterm and neovim configs as they are the only programs configured here that will also work on Windows
